# Setu

[猫猫酱](https://github.com/qzlzdy/Neko-Chan)色图库PC端源代码

## 编译

```bash
mkdir build
cd build
cmake ..
make
```

## 用途

`dbinit`初始化色图扩展名数据库

`pretty`用于在去除冗余色图后整理色图编号填补空缺

`encode`对未编号的色图编号
